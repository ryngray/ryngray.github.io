This event was created to build a welcoming environment where folks across identities can enjoy dancing, community, and a space to let loose!  We will be spanning the swing dances with this event, so come, learn to lindy or west coast and enjoy partner dancing!

# Date

Saturday, January 17th

# Lessons

Lindy Switch Lesson: 6:30-7:30, beginner-friendly lesson focusing on reducing the need for dance roles

West Coast Lesson: 7:30-8:30, beginner-friendly lesson focusing on fundamentals of west coast swing

$5 per lesson, collected money goes towards paying instructors.

No partner necessary!

# What to expect
We work to create a warm and welcoming environment to dance, connect, and be yourself! Expect to ask people to dance and to be asked to dance. Feel free to say no at any time and for any reason.

Check out [Santa Fe Swing Code of Conduct](https://santafebalboa.org/code-of-conduct/) for more details on our scene's code of conduct and dance ettiquette.

Please note that gender and gender expression do not correspond to dance roles!  Please communicate by asking whether someone would like to lead, follow, or switch for a given dance.  This can change between dances and between styles, so always ask!
